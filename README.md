# Rep4DJK_New

“教育大健康” 半自动打卡 - 重构Rep4DJK

## 说明

默认 `userData.json` 文件为空，需按照以下格式添加用户；


```
{
    "userId0": { "name": "userName0" },
    "userId1": { "name": "userName1" },
    "userId2": { "name": "userName2" }
}
```
其中 `userId` 为用户id, `userName`为用户名
| userId | userName |
| --- | --- |
| 99999 | BoiWong |
| 11111 | NewUser |
